# Sistema de Gestión de Estabilidad

Sistema para gestionar estudios de estabilidad de productos médicos.

## Características

- 📝 Ingreso de lotes con generación automática de períodos (1 a 60 meses)
- 🔍 Búsqueda por lote, producto, código, mes y año
- 📚 Catálogo de productos con autocompletado
- 📤 Exportación de datos a Excel
- ✏️ Edición de análisis
